AirLink

AirLink é uma ferramenta open-source de criação de páginas personalizadas de links, inspirada no LinkTree, mas totalmente gratuita. Permite aos usuários customizarem suas páginas com facilidade, sem custos, oferecendo uma experiência simples e eficiente para compartilhar links e conteúdos.

OnDemandCoursePlatform

Essa plataforma oferece cursos online com acesso baseado em diferentes níveis de assinatura (Free, Basic, Premium). Os usuários podem se cadastrar, fazer login seguro e acessar os cursos conforme a assinatura. Utilizando React para a interface e PostgreSQL para o banco de dados, a plataforma proporciona uma experiência de aprendizado prática e flexível.
